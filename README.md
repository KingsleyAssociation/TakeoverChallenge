🎮 The Takeover Challenge — Live Operational Arena
A youth event-planning simulation and pitch competition tool built for the Kingsley Association's seed grant program.

What This Is
The Takeover Challenge is an interactive, browser-based workshop disguised as a pitch competition. Youth crews are given a real $5,000 seed grant budget and challenged to plan, defend, and pitch an actual event — a one-night takeover, a recurring club night, a talent showcase, whatever they've got — in front of a facilitator panel acting as venue gatekeepers.

This isn't a form. It's an operational simulation. Every decision has a consequence. Budget shortcuts trigger coin flips. Missing adults get flagged. Same person on setup and cleanup? The system calls it out.

The tool runs entirely in-browser — no backend, no login, no setup. Open the HTML file and go.
